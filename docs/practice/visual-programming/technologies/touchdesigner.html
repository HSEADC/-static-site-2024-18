TouchDesigner — софт для работы с мультимедийным контентом. Позволяет создавать генеративную графику.

Что такое ноды?

Основным элементом в TouchDesigner являются ноды (или узлы), которые представляют собой блоки, выполняющие определённые функции. В этой статье мы подробно рассмотрим, что такое ноды, их типы и как они используются в процессе работы в TouchDesigner.

Настройки

У каждой ноды свои настройки Вопрос — информация о ноде из википедии Сине-желтый вопрос — информация о ноде по питону. Плюс — открывает все параметры нод Для редактирования мишень — показывает все параметры, которые были изменены
