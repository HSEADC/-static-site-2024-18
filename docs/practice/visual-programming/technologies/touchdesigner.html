Touchdesigner

Что это?

TouchDesigner — программа в которой используется нодовый язык визуального программирования для работы с мультимедийным контентом

Нода?

Нода — это контейнер с информацией, который соединяется с другими нодами, образуя цепочку, в которой выстраивается последовательность действий

Параметры нод

У каждой ноды свои настройки

  • Вопрос — информация о ноде из википедии
  • Сине-желтый вопрос — информация о ноде по питону
  • Плюс — открывает все параметры нод
  • Для редактирования мишень — показывает все параметры, которые были изменены

Сигналы

  • Красный крест — ошибка нода не работает
  • Желтый восклицательный знак — предупреждения об ограничениях
